Official jPlayer Fabrica Plugin
===============================

A sharp, elegant and customizable audio player that will set your audio apart.

Installation
============

1. Copy the wp-jplayer-fabrica folder into the wp-content folder within your Wordpress installation. 
2. Visit the plugins page and activate Official jPlayer Fabrica Plugin.

How to Use
==========

This plugin works by using Wordpress Plugin Shortcodes.

For example once you've activated the plugin, add the following to your page to create a stylish audio player:
[jplayer-fabrica src="http://jplayer.org/audio/mp3/Miaow-01-Tempered-song.mp3" width="400px" primaryColor="#ffffff" secondaryColor="#622181" startColor="#ea6338" stopColor="#e03288" ]My song title goes here[/jplayer-fabrica]

The following values are supported:

    src - the src URL (address) of your media file*
    width - the width in px (pixels) or percent (%) you want your player to be
    primaryColor - your button background, progress bar and text color
    secondaryColor - your button icon and progress bar background color
    startColor - your gradient fill start color
    stopColor - your gradient fill stop color
    add your song title between the [jplayer-fabrica ...] and [/jplayer-fabrica] tags 

Note: Sensible default values are provided.

*If you have uploaded media to your Wordpress site you can find the File URL in the right-hand panel when you edit the media from the Media page.
